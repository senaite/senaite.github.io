Next Generation LIMS

The SENAITE Project provides a complete new and modern way to interact with Bika LIMS.

Visit the GitHub Project Page

"SENAITE is a modern, mobile first version of the web based open source LIMS Bika."

Features

  • Clean and Responsive User Interface
  • Full JSON API to interface core Bika LIMS functionality
  • Integrates modern Web Frameworks
  • Fully Open Source

Specifications

  • Compatible with the latest Bika LIMS version
  • Built on Plone CMS
  • Integrates Twitter Bootstrap Framework
  • Rich UI components built with ExtJS

SENAITE is a beautiful trigonal, oil-green to greenish black crystal, with almost the hardness of a diamond. Although the crystal is described with a complex formula, it still has clear and straight shapes. Therefore, it reflects nicely the complexity of the LIMS, while providing a modern, intuitive and friendly UI/UX.
